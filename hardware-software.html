Hardware & Software

Learn about physical components and programs used in computers.

What is Hardware?

Hardware refers to the physical parts of a computer that can be seen and touched.

  • Keyboard
  • Mouse
  • Monitor
  • CPU
  • Printer

What is Software?

Software is a set of programs and instructions that tell a computer what to do.

Types of Software

1. System Software

  • Operating System
  • Device Drivers
  • Utility Programs

2. Application Software

  • MS Word
  • MS Excel
  • Google Chrome
  • Photoshop

Difference Between Hardware and Software

Hardware Software
Physical Components Programs & Instructions
Can be touched Cannot be touched
Examples: Keyboard, Monitor Examples: Windows, MS Word
