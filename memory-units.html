Memory Units

Learn about bits, bytes, KB, MB, GB and TB.

What is Memory?

Memory is used to store data and instructions in a computer.

Basic Memory Units

  • 1 Bit = Binary Digit (0 or 1)
  • 1 Byte = 8 Bits
  • 1 KB = 1024 Bytes
  • 1 MB = 1024 KB
  • 1 GB = 1024 MB
  • 1 TB = 1024 GB

Types of Memory

  • Primary Memory (RAM, ROM)
  • Secondary Memory (Hard Disk, SSD, Pen Drive)
