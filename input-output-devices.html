Input & Output Devices

Learn about devices used to enter data and receive information.

What are Input Devices?

Input devices are used to enter data and instructions into a computer.

  • Keyboard
  • Mouse
  • Scanner
  • Microphone
  • Webcam

What are Output Devices?

Output devices display the processed information from a computer.

  • Monitor
  • Printer
  • Speaker
  • Projector
  • Plotter

Examples

Keyboard and mouse are input devices, while monitor and printer are output devices.
